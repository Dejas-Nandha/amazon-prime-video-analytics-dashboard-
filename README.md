## Project Overview
This project features an interactive Power BI dashboard built using a publicly available Amazon Prime Video dataset. The dashboard analyzes movies and TV shows to uncover insights related to content type, ratings, genres, directors, and release trends over time.

## Dataset Description
The dataset used in this project contains information about Amazon Prime Video titles with the following attributes:
- Show ID
- Content Type (Movie / TV Show)
- Title
- Director
- Cast
- Country
- Date Added
- Release Year
- Rating
- Duration
- Genre (Listed In)
- Description
